IF CLAUSES
If Clause Type 0
Doğruluğu hiçbir zaman değişmeyecek, her zaman geçerli olan olasılıklardan bahsedildiğinde kullanılır.
If Clause Type 1
Şu ana ait olasılıklardan bahsedildiğinde kullanılır.
If Clause Type 2
Geçmiş ve şimdiki zaman birlikte kullanılır, fakat şu ana ait olasılığı ifade eder.
If Clause Type 3
Mişli geçmiş zaman ile birlikte kullanılır ve geçmişe ait olasılığı ifade eder.
